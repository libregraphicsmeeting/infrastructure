# Information for local organizers

## Information about the LGM

We tend to store the information in Git repository.

Currently, the repositories are _stored_ on Github (being migrated to Codeberg).

Local organizers are welcome to create and store documents in any way they see fit, but it would be nice if the final _result_ could be stored in one of the existing repositories (or a newly created one, for new type of content).

## Website

The yearly websites are published as static sites on KDE infrastructure by producing the Html files from an ad-hoc repository on https://invent.kde.org/

- Each LGM edition has websites that is placed at libregraphicsmeeting.org/20xx. The KDE administrator deals with setting up the redirection to the proper yearly website.
- The sources for the website can be anywhere you wish. If you don't have a specific preference, we can setup a repository on Codeberg. The local organizer owns the repository.
- The KDE admin creates a repository that will be used to generate the HTML files that are automatically pushed to the webserver.
- The local organizer has write access to the KDE repository.
- The KDE repository can contain instructions for generating the HTML from sources.
- After the LGM has ended, the local organizer puts the website into a state that is meant to be the archived version.
- As soon as the next organizer has a new website ready and we estimate that the interest for looking up the next LGM is bigger than looking at the last one, the next website becomes the default one.

## Communcation channels

The LGM has multiple communication tools that can be used by the local organizer. 

- Messaging: we have a Matrix channel and an IRC channel that are kept in sync by KDE infrastructure.
- Mailing list: we have multiple mailing lists that can be  used:
  - A general mailing list for general that used to be the main communication channel for the LGM but as of 2025 does not have any traffic (but 124 subscribers).
  - The org mailing list for the people from the community being involved in the LGM organisation (not in use for several years now)
  - The local org mailing list (never been used; most local organizers already have a preferred communication channel to keep in touch with the local helpers)
- A Mastodon account with 150 followers (at the time of writing, actively in use)
- The website
- A Facebook account (not in use since 2020)
- A Youtube channel with most (all?) the recordings
- Multiple _freeer_ alternative videos platforms that are being consolidated into the CCC TV infrastructure.
